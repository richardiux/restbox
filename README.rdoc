= Restbox

== Install

To install as a Rails plugin, run this command.

  script/plugin install http://github.com/richardiux/jQuery.restbox(.git

=== Prerequisites
  JQuery 1.3.2

== Setup

in your layout after including jQuery:

  <%= javascript_include_tag 'jQuery.restbox(' %>
  <%= stylesheet_link_tag 'jQuery.restbox(' %>

== Example on Rails Application

=== In your application.js

  $(document).ready(function() {
    jQuery("a.jQuery.restbox(").jQuery.restbox(({})
  }

=== In your view

  <%= link_to "edit", order_path(@order), :class => "jQuery.restbox(" %>
  

=== In your controller

  def edit
    @order = Order.find params[:id]
    respond_to do |format|
      format.html { }
      format.js   { }
    end
  end
  
=== The JS view for the action (edit.js.erb)

You can use the following javascript helpers

To put text in the jQuery.restbox(:
  <%= jQuery.restbox((:content => 'some text') %>

To put a partial inside the jQuery.restbox(:
  <%= jQuery.restbox((:content => render(:partial => 'form')) %>


== With javascript

To put content in the jQuery.restbox(
  $.jQuery.restbox(({content: "some text"})

To manually display loading jQuery.restbox(
  $.jQuery.restbox(({})
  
== Upgrading

=== Reinstall the plugin
 script/plugin install http://github.com/richardiux/jQuery.restbox(.git -f
